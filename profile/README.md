# Citystroom

Urban intelligence platform for city governments — heat, air, energy and people flows on one map.

**Products**
- **Stroom Live** — real-time city dashboard
- **Stroom Predict** — 72-hour urban forecast engine
- **Stroom API** — data access for planners and researchers

**Stack:** Python · Rust · NETCDF · OpenFOAM · VTK

---

We build tools for cities. Some of them live here — CFD viewers, urban data utilities and Stroom API examples.
